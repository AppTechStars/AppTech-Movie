# Movie Application

This is a simple movie application built with React and Vite. It displays a list of movies with their titles, posters, and descriptions.

## Project Structure

```
movie-app
├── src
│   ├── components
│   │   └── MovieCard.jsx
│   ├── pages
│   │   └── Home.jsx
│   ├── App.jsx
│   ├── index.jsx
│   └── styles
│       └── App.css
├── public
│   └── index.html
├── package.json
├── .gitignore
└── README.md
```

## Getting Started

To get started with the movie application, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd movie-app
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the application.

## Features

- Displays a list of movies using the `MovieCard` component.
- Responsive design with CSS styles for a better user experience.
